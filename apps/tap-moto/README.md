# 别惹小毛头

以小毛头（MOTO）为主角的手机端聚会 H5 游戏。多人轮流点击 16 个小毛头，点中唯一的暴怒小毛头即判负并抽取挑战卡。

## 特性

- 4×4 随机踩雷棋盘
- 安全表情、离场动画和快速音效
- 暴怒演出、屏幕震动与设备短震
- 24 张会话内尽量不重复的挑战卡
- 无登录、无数据库、无业务网络请求
- 适配手机竖屏和安全区

## 本地运行

需要 Node.js 22.13 或更高版本。

```bash
npm ci
npm run dev
```

打开终端显示的本地网址即可游玩。

## 构建与测试

```bash
npm run build
npm test
```

## 静态导出

本项目已配置为合集网站中的 `games/tap-moto/` 子页面。仓库根目录的 GitHub Pages 工作流会自动设置正确路径并生成静态文件。

需要单独验证静态导出时，请设置 `STATIC_EXPORT=true`、`NEXT_PUBLIC_BASE_PATH` 和站点来源 `SITE_URL`，再运行 `npm run build:static`。输出目录为 `out/`。

## 主要目录

- `app/`：游戏逻辑、页面和样式
- `public/assets/`：小毛头角色素材
- `tests/`：随机位置、挑战卡与页面构建测试
