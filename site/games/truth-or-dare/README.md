# 真心话大冒险

无需安装依赖的手机端静态网页游戏，包含 42 张真心话卡和 50 张大冒险卡。当前作为 CRUSH 聚会游戏合集的 `games/truth-or-dare/` 子页面发布。

## 玩法

- 点击“真心话”或“大冒险”抽取对应卡牌。
- 点击“都可以”进行加权随机抽取；当前权重为 25% 真心话、75% 大冒险。
- 卡牌抽完一轮后会自动重新洗牌，同一轮内不会重复。
- 可点击关闭按钮、点击遮罩或按 Esc 关闭卡牌。

## 本地运行

建议在本目录启动一个静态文件服务器：

```bash
python3 -m http.server 8000
```

然后在浏览器中打开 `http://localhost:8000`。

合集网站的发布方式请参阅仓库根目录 `README.md`。

## 主要目录

- `index.html`：游戏入口
- `css/`：页面样式
- `js/`：抽卡、洗牌、翻牌和音效逻辑
- `images/`：背景、Logo、卡背和随机按钮
- `truth-cards/`：42 张真心话卡面
- `dare-cards/`：50 张大冒险卡面
- `audio/`：抽卡音效

## 使用说明

建议使用最新版 Safari、Chrome 或 Edge。涉及饮酒或挑战的内容均应以自愿、安全和遵守所在地法律为前提，可随时跳过或改用无酒精饮品。
