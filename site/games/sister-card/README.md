# 大小姐牌

无需安装依赖的静态网页游戏。当前作为 CRUSH 聚会游戏合集的 `games/sister-card/` 子页面发布。

## 玩法

- 拖动牌堆旋转浏览，点击或向上拖动牌面即可抽牌。
- 点击中央已抽出的牌可将其收起。
- 点击右上角“重新洗牌”开始新一轮。
- 支持键盘方向键浏览，并可用 Enter 或空格键抽牌。

合集网站的发布方式请参阅仓库根目录 `README.md`。

## 本地打开

可以直接打开根目录的 `index.html`。手机或 iPad 访问时，建议在本目录启动本地服务器：

```bash
python3 -m http.server 8000
```

然后打开 `http://localhost:8000`。

## 使用说明

建议使用最新版 Safari、Chrome 或 Edge。游戏包含饮酒主题，但可用无酒精饮品代替；请遵守所在地法律并理性参与。

本仓库当前未附开源许可证，公开可见不代表授权他人复用。公开发布前，请确认其中的图片、音频和视频素材均为原创或已取得公开再分发许可。

## 目录

- `index.html`: 游戏入口
- `css/`: 页面样式
- `js/`: 牌库、抽牌、动效和音效逻辑
- `images/`: 背景、牌面和牌背
- `animations/`: 特殊牌动效
- `audio/`: 抽牌及特殊牌音效
